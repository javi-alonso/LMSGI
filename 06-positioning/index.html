Positioning en CSS

¿Qué es la propiedad position?

La propiedad position define cómo se posiciona un elemento en el flujo del documento y cómo responde a las propiedades top, left, right y bottom.

Valores principales

  • static: posición por defecto, sigue el flujo normal.
  • relative: se posiciona en relación a su posición original.
  • absolute: se posiciona respecto al primer contenedor con posición distinta de static.
  • fixed: se fija en la ventana del navegador (viewport).
  • sticky: combina el comportamiento de relative y fixed.

Ejemplo 3: position: absolute

Un elemento con position: absolute se posiciona en relación al primer contenedor padre que tenga una posición distinta de static.

En este ejemplo, el cuadro morado está dentro de un contenedor gris que tiene position: relative;. Se posiciona con top: 20px; y right: 20px;.

Absoluto
Contenedor (relative)
.absolute-container {
  position: relative;
}
.abs {
  position: absolute;
  top: 20px;
  right: 20px;
}

Ejemplo 5: position: sticky

Se comporta como relative hasta alcanzar un punto, donde se fija.

  • El contenedor tiene scroll vertical (overflow-y: auto) y una altura limitada.
  • El encabezado dentro del contenedor tiene position: sticky y una propiedad top definida.
  • Cuando haces scroll dentro del contenedor, el encabezado se desplaza hasta que su borde superior alcanza el valor definido por top.
  • En ese punto, el encabezado se "pega" y permanece visible mientras el resto del contenido sigue desplazándose.
