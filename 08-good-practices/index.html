Buenas prácticas en CSS

¿Por qué seguir buenas prácticas?

Las buenas prácticas en CSS ayudan a que tus estilos sean más fáciles de mantener, más rápidos y más coherentes entre proyectos. Aquí verás una lista de consejos prácticos con ejemplos visuales.

Usa variables CSS

Las variables permiten mantener consistencia en colores, fuentes y tamaños.

:root {
  --color-primary: #2b6cb0;
  --radius: 8px;
}
button {
  background: var(--color-primary);
  border-radius: var(--radius);
}

Usa unidades relativas

Las unidades relativas (em, rem, %, vh, vw) se adaptan mejor a diferentes pantallas.

rem / %
px

Usa clases descriptivas y evita selectores excesivos

Evita selectores largos o dependientes del HTML. Usa nombres de clase claros y reutilizables.

/*Malo */
main article section div ul li a span { color: red; }

/*Bueno */
.nav-link { color: red; }

Mobile First y Media Queries claras

Diseña primero para pantallas pequeñas y luego amplía con min-width.

.container {
  font-size: 1rem;
}
@media (min-width: 768px) {
  .container {
    font-size: 1.2rem;
  }
}
