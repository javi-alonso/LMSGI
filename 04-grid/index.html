CSS Grid Layout

¿Qué es CSS Grid?

CSS Grid es un sistema de diseño bidimensional que permite organizar los elementos en filas y columnas, proporcionando control total sobre la disposición en una cuadrícula.

Sintaxis básica

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 1rem;
}
.item {
  background: #2b6cb0;
  color: white;
  text-align: center;
  padding: 1rem;
}

Definir filas y columnas

Podemos definir el número y tamaño de filas y columnas con grid-template-columns y grid-template-rows. También podemos usar funciones como repeat() y unidades fr (fracciones del espacio disponible).

grid-template-columns: 1fr 2fr 1fr;

Grid Areas

Podemos asignar nombres a las zonas de la cuadrícula con grid-template-areas, y luego usar grid-area en cada elemento.

.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 1fr 2fr;
  gap: 0.5rem;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

Alineación y justificación

Con justify-items y align-items puedes alinear los elementos dentro de sus celdas.

justify-items: center; align-items: center;
