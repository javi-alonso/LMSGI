Animaciones en CSS

¿Qué son las animaciones?

Las animaciones en CSS permiten modificar de forma gradual los valores de las propiedades de un elemento a lo largo del tiempo. Se definen con @keyframes y se aplican mediante la propiedad animation.

Sintaxis básica

@keyframes nombreAnimacion {
  from { propiedad: valorInicial; }
  to   { propiedad: valorFinal; }
}

.elemento {
  animation-name: nombreAnimacion;
  animation-duration: 2s;
  animation-iteration-count: infinite;
  animation-timing-function: ease-in-out;
}

Propiedades principales de animación

  • animation-name: Nombre del bloque @keyframes que se aplicará.
  • animation-duration: Duración total de la animación (por ejemplo, 2s o 500ms).
  • animation-timing-function: Curva de aceleración (por ejemplo, linear, ease-in, ease-out).
  • animation-delay: Tiempo de espera antes de empezar la animación.
  • animation-iteration-count: Número de repeticiones (infinite para bucles).
  • animation-direction: Permite invertir la animación en cada repetición (alternate).
  • animation-fill-mode: Controla el estado final (forwards, backwards).

Ejemplos visuales

Movimiento lineal

animation: moveRight 2s infinite alternate ease-in-out;

Cambio de color

animation: colorChange 3s infinite alternate;

Rebote (bounce)

animation: bounce 1.5s infinite ease-in-out;

Rotación

animation: spin 4s linear infinite;

Múltiples animaciones

animation: moveRight 2s infinite alternate ease-in-out, colorChange 3s infinite alternate;

Animaciones al pasar el ratón (hover)

Las animaciones también pueden activarse al pasar el ratón sobre un elemento. Esto se consigue combinando :hover con propiedades animation o transition.

animation: pulse 0.8s ease-in-out; (solo al hacer hover)

Buenas prácticas

  • Evita animar propiedades que afecten al layout (como width o height); usa transform o opacity para mayor rendimiento.
  • Usa alternate y ease-in-out para animaciones suaves.
  • Define duraciones coherentes (entre 0.3s y 3s normalmente).
  • Si la animación es decorativa, usa prefers-reduced-motion para respetar la accesibilidad.
