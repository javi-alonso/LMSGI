Estructura básica de un documento HTML

¿Qué es la estructura de HTML?

Todo documento HTML tiene una estructura básica que define cómo se organiza el contenido de una página web. Esta estructura permite a los navegadores interpretar correctamente la información.

Estructura general:

<!doctype html>
<html lang="es">
  <head>
    <meta charset="utf-8">
    <title>Título de la página</title>
  </head>
  <body>
    <header>Encabezado</header>
    <main>Contenido principal</main>
    <footer>Pie de página</footer>
  </body>
</html>

Partes principales:

  • DOCTYPE: Indica al navegador que el documento es HTML5.
  • <html>: Raíz del documento. Contiene todo el código HTML.
  • <head>: Contiene metadatos, enlaces a CSS, título, etc.
  • <body>: Contiene todo el contenido visible en la página.
