Accesibilidad Web (A11y)

¿Qué es la accesibilidad web?

La accesibilidad web (A11y) busca que todas las personas, incluyendo aquellas con discapacidades visuales, auditivas, motrices o cognitivas, puedan usar y entender un sitio web.

En CSS y HTML, aplicar accesibilidad significa cuidar el contraste, las fuentes, el foco, los textos alternativos y la navegación por teclado.

Contraste de colores

El texto debe tener suficiente contraste con su fondo para ser legible. Usa herramientas como Contrast Ratio para verificarlo.

Texto alternativo en imágenes

Las imágenes deben tener alt para describir su contenido a los lectores de pantalla.

Tamaños de texto accesibles

Usa unidades relativas (em o rem) para que el texto se adapte al tamaño configurado por el usuario.

Animaciones accesibles

Las animaciones deben ser suaves y ofrecer la opción de reducir movimiento con @media (prefers-reduced-motion: reduce).

Buenas prácticas generales

  • Usa etiquetas semánticas: <header>, <main>, <nav>, <footer>.
  • Usa aria-label y roles solo cuando no haya alternativa semántica.
  • Evita texto dentro de imágenes: usa HTML + CSS siempre que sea posible.
  • Prueba la navegación por teclado y lectores de pantalla.
