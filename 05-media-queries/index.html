Media Queries en CSS

¿Qué son las Media Queries?

Las Media Queries permiten aplicar estilos CSS diferentes en función de las características del dispositivo, como el ancho de la pantalla, la orientación o la resolución. Son esenciales para crear diseños responsivos.

Sintaxis básica

@media (max-width: 768px) {
  body {
    background-color: lightblue;
  }
}

Esto significa: “Cuando el ancho de la pantalla sea de 768 píxeles o menos, aplica estos estilos”.

Reglas aplicadas:

  • En pantallas grandes: 4 columnas
  • En tablets: 2 columnas
  • En móviles: 1 columna

Buenas prácticas

  • Usa min-width (mobile-first) siempre que sea posible.
  • Aplica puntos de ruptura (breakpoints) en función del contenido, no del dispositivo.
  • Evita usar tamaños fijos en píxeles: usa em, rem o %.
  • Combina Media Queries con Flexbox o Grid para una adaptación fluida.
