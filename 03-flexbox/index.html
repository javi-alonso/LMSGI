Flexbox en CSS

¿Qué es Flexbox?

Flexbox es un modelo de diseño en CSS que facilita la distribución, alineación y adaptación de los elementos dentro de un contenedor flexible.

Se activa con display: flex; en el contenedor, y los elementos hijos se vuelven “ítems flexibles”.

Estructura básica

.container {
  display: flex;             /* Activa Flexbox */
  justify-content: center;   /* Alinea en el eje horizontal */
  align-items: center;       /* Alinea en el eje vertical */
  gap: 1rem;                 /* Espacio entre elementos */
}
.item {
  background: #2b6cb0;
  color: white;
  padding: 1rem;
  border-radius: 8px;
}

Propiedades de los elementos hijos

Determina cuánto crece cada elemento para ocupar el espacio sobrante.

flex-shrink

Determina cuánto puede reducirse cada elemento si el contenedor es más pequeño.

Permite sobrescribir align-items para un ítem concreto.
