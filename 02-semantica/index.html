Semántica en HTML

¿Qué significa la semántica en HTML?

La semántica en HTML se refiere al uso de etiquetas que describen claramente el contenido que contienen. Esto ayuda a los navegadores, motores de búsqueda y tecnologías asistivas a interpretar correctamente la información.

Ejemplos de etiquetas semánticas:

  • <header> → Encabezado de la página o sección.
  • <nav> → Bloque de navegación con enlaces.
  • <main> → Contenido principal del documento.
  • <article> → Contenido independiente (una noticia, post, etc.).
  • <section> → Agrupa contenido temático.
  • <aside> → Contenido complementario (barras laterales, notas).
  • <footer> → Pie de página.

Estructura semántica típica:

<header>
  <h1>Título del sitio</h1>
  <nav>
    <a href="#">Inicio</a>
    <a href="#">Artículos</a>
    <a href="#">Contacto</a>
  </nav>
</header>

<main>
  <article>
    <h2>Título de artículo</h2>
    <p>Contenido principal...</p>
  </article>
</main>

<footer>
  <p>Mi sitio web</p>
</footer>
